1. Status vom Projekt

	- C4H-APP is in der Fortgeschrittener Testphase

2. Benutzung der Software

	- Die software startet automatisch nach dem Neutsart oder nach der Installation.

3. Allgemeine Installationshinweise:

 3.1 Setup

	- setup_c4H ausführen 

 3.2 Deinstallationshinweise

	- unistall000.exe 

4. Installationshinweise 
 
	- Das Programm startet ausschließlich mit unseren 3S- Musterimage (Windows 10, Windows 7, x64)
	- Das Programm startet automatisch nach dem Neustart, alternativ direkt nach der Installation.
	- Ein Registry- Eintrag wird für AllUser erstellt. 
	- Ein Javaw Process ist dann aktiv und sorgt für den Verlauf der App. 
	- Eine Netzwerkverbindung ist eine Voraussetzung für den integrierten Browser.

5. License

	- Das C4H-APP Projekt basiert auf Open Source Quellen. 
	- Es wird die Java Version 7 für das Auslesen der Netzwerkdaten verwendet. 
	- Die aktuelle Version 3.4 von Bootstrap wird für den Browswer verwendet und dient zur Darstellung der Eingabemaske.
